# 🔰 R Fundamentals

> Core R programming concepts — variables, vectors, data frames, data types, and base plotting — built through hands-on lab exercises and reference scripts.

---

## 📂 Projects

| Folder | Description | Key Skills |
|--------|-------------|------------|
| [r-intro-labs](r-intro-labs/) | Four progressively complex labs covering R basics from variable assignment through descriptive statistics | Variables, vectors, data frames, factors, Z-scores, `na.rm` |
| [base-plots](base-plots/) | Base R plotting reference using Fibonacci and `msleep` data | `plot()`, `hist()`, `boxplot()`, `stem()`, `pch`, `lty`, `cex` |

---

## 🛠️ Packages Used

| Package | Purpose |
|---------|---------|
| `tidyverse` | Loads `ggplot2` for the `msleep` dataset; `readr` for data import |
| `airports` | Provides the `usairports` dataset used in Lab 2 |
| `readxl` | Reading `.xlsx` files in Lab 3 |

---

## 📋 Concepts Covered

- Assigning and manipulating variables and vectors
- Indexing, slicing, and modifying R data structures
- Building data frames from scratch with `data.frame()` and `cbind()`
- Type conversion with `as.factor()`, `as.data.frame()`, `class()`
- Data type classification: nominal, ordinal, interval, ratio, discrete, continuous
- Descriptive statistics: `mean()`, `median()`, `var()`, `sd()`, `summary()`
- Handling `NA` values with `na.rm = TRUE`
- Z-score calculation by hand
- Base R plotting: scatter, histogram, stem-and-leaf, boxplot
- Plot customization: titles, labels, color, point shape, line type
